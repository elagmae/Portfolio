STEREOGAME

Concept

Jeu web multi-platform en 2D pixel-art ou le joueur incarne un régisseur de scène de Stereolux, qui doit trouver des tableaux comportants les infos de l'évènement à promouvoir. Il se joue sur 7 niveaux, pour chacun des jours de l'évènement. Pour rendre le tout plus intéressant, il est également possible de récupérer quelques easter eggs...

Pourquoi ?

Ce projet était une mission professionnelle accordée par Stereolux. Avec cette mission, moi et mon équipe avions l'objectif de promouvoir leur évènement Stereogame de manière simple, facile d'accès et surtout extrêmement casual, pour que n'importe qui puisse y jouer et apprécier un univers correspondant à Stereolux.

Mon travail

  • Création du concept.
  • Intégration des sons et code d'un Audio Manager avec FMOD
  • Création d'effets visuelles avec Dotween, les Particules Systems, et le code d'animations sur ou sans assets.
  • Utilisation du support web multi-platform pour les inputs du joueur.
